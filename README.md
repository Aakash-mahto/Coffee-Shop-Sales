# Coffee-Shop-Sales
☕ Coffee Shop Sales Analysis Dashboard
📌 Project Overview

This project is an interactive Coffee Shop Sales Dashboard built using Microsoft Excel, Power Pivot, and DAX. The objective is to analyze coffee shop sales data, uncover business insights, and visualize key performance indicators (KPIs) to support data-driven decision-making.

The dashboard enables users to filter data by Month and Day using slicers, providing an interactive experience for exploring sales trends and customer behavior.

🚀 Features
Interactive dashboard with slicers
Monthly and daily sales filtering
KPI cards for business performance
Sales trend analysis
Product performance analysis
Store location comparison
Category-wise sales distribution
Order analysis by weekday
Quantity analysis based on business hours
📊 Key Performance Indicators (KPIs)
💰 Total Sales
👥 Total Footfall
🧾 Average Bill per Person
☕ Average Orders per Person
📈 Dashboard Insights
Sales Performance
Tracks overall revenue generated by the coffee shop.
Displays total customer footfall and average spending.
Product Analysis
Identifies the Top 5 Best-Selling Products.
Helps understand customer purchasing preferences.
Category Analysis
Shows percentage contribution of each product category.
Identifies high-performing categories.
Time-Based Analysis
Quantity ordered by hour.
Peak business hours.
Weekly order trends.
Store Performance
Compares sales and customer footfall across multiple store locations.
Helps identify the best-performing branch.
